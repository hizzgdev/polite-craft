# Contributing to Polite Craft

Thank you for your interest in contributing.

## Setup

```bash
git clone https://gitee.com/hizzgdev/polite-craft.git
# 或 git clone https://github.com/hizzgdev/polite-craft.git
cd polite-craft
npm install
npm run build
```

Load the extension from the **`dist/`** directory at `chrome://extensions/` (Developer mode → Load unpacked).

## Making changes

1. Edit files under **`src/`** only — `dist/` is generated by `npm run build`.
2. Run tests before opening a PR:

   ```bash
   npm test
   ```

3. If you change `background.js` or `content.js`, also run `npm run build` and smoke-test on a GitHub PR/Issue page.

## Pull requests

- Keep diffs focused; one logical change per PR when possible.
- Do not commit API keys, `.env` files, or captured authenticated web pages.
- New `host_permissions` or `permissions` must be explained in the PR description.

## Reporting issues

Include Chrome version, extension version (`manifest.json`), and steps to reproduce. For GitHub DOM injection bugs, note whether the page is a PR or Issue and attach a screenshot of the comment composer if helpful.
